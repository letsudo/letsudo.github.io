搭建技术博客 Hexo + Github in MacOS

Posted on 2019-05-02 | In 工具学习

本文主要是记录在macos上配置搭建 hexo + github的技术博客过程，便于后期回顾
内容来自互联网，如若侵权请联系我删除

一、介绍

hexo + github搭建的技术博客环境，整个流程类似于 将编写的md文件与hexo的主题和网站的设置，通过node.js生成 静态网页，并部署到github上

Hexo

一个把Markdown等文档生成静态网页的博客框架
Hexo官网

github

全球最大的同性社交网站
github官网

node.js npm nvm 关系

  • node.js 构建网页
  • npm: 是随同node.Js一起安装的包管理工具，npm管理对应node.Js的第三方插件
  • nvm: node.Js版本管理工具,管理node.js版本和npm版本

三、 切换主题

我选定的主题是Next
配置方法可参见官网教程
 
在 Hexo 中有两份主要的配置文件，其名称都是 _config.yml。

  • 一份位于站点根目录下，主要包含 Hexo 本身的配置；
  • 另一份位于主题目录下，这份配置由主题作者提供，主要用于配置主题相关的选项。
    为了描述方便，在以下说明中，将前者称为 站点配置文件， 后者称为 主题配置文件

下载

建议你使用 克隆最新版本 的方式获取，主题将放在themes文件夹下

启用主题

打开站点配置文件 ,找到 theme 字段，并将其值更改为 next

修改后最好clean下缓存

此时可以hexo s在本地测试下效果 此时即可使用浏览器访问 http://localhost:4000，检查站点是否正确运行

添加标签

  2. 设置页面类型：修改index.md文件, 修改type 为 tags,注意冒号后要有空格

  3. 在主题配置文件中添加tag到menu

这样之后在md文章中添加tag即可实现分类,分类、归纳等界面相同
