Markdown语法总结

Posted on 2019-05-02 | In 工具学习

本文主要记录学习Markdown语法的总结，便于后期查阅温习
内容来自互联网，如若侵权请联系我删除
p.s. 部分语法某些平台可能不支持

一、 概念

Markdown

我的理解是: Markdown通过简单语法使纯文本格式化，便于读写

二、语法

1. 标题

  • 类 Setext 形式, 其中 = 和 - 可以任意数量

  • 类 Atx 形式 行首插入 1 到 6 个 # ，支持六级标题
